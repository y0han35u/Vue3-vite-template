# vue3-test-project
This is test project 
for vue3 practice.

## run docker
```
docker-compose up -d --build
docker_compose exec vue3 /bin/ash
```

## stop docker
```
docker-compose down
```

## start project
```
cd / && npm init vite-app app && cd /app && npm install
```

## run server
```
npm run dev
```
